# EFN Utility Adapter - C# SDK

A .NET 8 Minimal API implementation of the EFN Utility Provider Adapter.

## Quickstart
1. Set your secret: `export EFN_API_SECRET="your_secret"`
2. Run the server: `dotnet run`

## Customization
Open `Program.cs` and implement your logic inside the `app.MapPost(...)` handlers.
