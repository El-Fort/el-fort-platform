# EFN Utility Provider Adapter SDK

Plug-and-play SDKs for Utility Companies (Electricity, Water, Cable TV, etc.) to integrate with the El-Fort Payment Network.

## Quickstart
1. Choose your language.
2. Search for `TODO` in the main file and implement your database lookup/token generation logic.
3. Deploy and provide EFN with your Base URL.

## Supported Languages
- Python (FastAPI)
- Node.js (Express)
- Go (Gin)
- Rust (Axum) - coming soon
- Java (Spring Boot) - coming soon
- C# (.NET Core) - coming soon
